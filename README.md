# The Rama Vision

Professional photo booth rental service website hosted on GitHub Pages.

## Overview

The Rama Vision provides premium photo booth rental services for events including weddings, corporate events, parties, and special occasions. This website serves as the primary online presence and booking platform.

## Website Structure

- **index.html** - Homepage with service overview and introduction
- **packages.html** - Photo booth rental packages and pricing
- **faqs.html** - Frequently asked questions
- **contact.html** - Contact form and business information
- **styles.css** - Custom styling and responsive design
- **script.js** - Interactive features and navigation

## Local Development

To preview changes locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/bllindhandsrightfast29/the-rama-vision.git
   cd the-rama-vision
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   python3 -m http.server 8000
   # Visit http://localhost:8000
   ```
